## Installation
1. Copy UnityNEAT to Application.persistentDataPath
- Windows: C:\Users\<user>\AppData\LocalLow\DefaultCompany\
- other Platform: https://docs.unity3d.com/ScriptReference/Application-persistentDataPath.html
2. run exe
